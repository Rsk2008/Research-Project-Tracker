## Push to your GitHub
1. Create a new empty repo on GitHub (without README).
2. In this folder run:
```bash
git init
git add .
git commit -m "feat: initial full-stack submission"
git branch -M main
git remote add origin YOUR_REPO_URL
git push -u origin main
```
